Frequently Asked Questions

What API Key do I need to use MixBanana?
You need an OpenRouter API Key. We act as a secure interface for the Gemini 3 Pro model. This ensures you pay the lowest direct provider rates without any markup from us.
Is my data secure with BYOK?
Yes. "Bring Your Own Key" means we never store your credentials on our servers. Your API key stays locally on your device during the session and is sent directly to the OpenRouter API securely.
Can I use generated images for commercial ads?
Absolutely. Since you are using your own API key, you own the commercial rights to the assets you generate, subject to OpenRouter and Google's standard commercial terms for the Gemini model.
What is "Nano Banana Pro"?
It is our specialized implementation of the Gemini 3 Pro Image Preview model. We have fine-tuned the prompt engineering layer to ensure outputs adhere specifically to high-conversion marketing aesthetics.
